# GWAS Data Directory

이 폴더는 파킨슨병 GWAS summary statistics 파일을 저장합니다.

## 파일 설명

- `GCST009325.h.tsv.gz`: 파킨슨병 GWAS 요약통계 (Nalls et al. 2019)
  - 17.4M SNPs
  - 37,688 케이스 + 1.4M 컨트롤
  - hg19 좌표계

## 데이터 출처

- **연구**: Nalls et al. (2019) The Lancet Neurology
- **DOI**: 10.1016/S1474-4422(19)30320-5
- **다운로드**: GWAS Catalog (GCST009325)
